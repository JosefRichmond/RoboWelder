Adding New Icons to Library

You are able to add new icons in addition to preinstalled.
1. Open icons library
You can do it from IconsApp by pressing the Open Library button (also you can use CLI).
Library folder has a path <add-on root>/icons
2. Add your icons to library
Several icons you can add to MyIcons folder
If you are going to add a whole icon pack add a folder containing your pack to the library folder.
3. Restart IconsApp to update the library
